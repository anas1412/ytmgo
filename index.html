Installation

Up and running
in 30 seconds.

Install dependencies, build, and go. It's that simple.

1

Install dependencies

You need mpv for playback and yt-dlp for search/download. Optionally install Brave/Firefox/Chrome for cookie extraction.

2

Build the binary

One command compiles everything into a single binary.

3

Run it

Launch ytmgo and start listening. That's it.

# Debian / Ubuntu
sudo apt install mpv yt-dlp

# macOS
brew install mpv yt-dlp

# Clone & build
git clone https://github.com/anas1412/ytmgo.git
cd ytmgo
go build -o ytmgo .

# Run
./ytmgo
